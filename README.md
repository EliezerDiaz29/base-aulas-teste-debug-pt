# Calculadora - Testes Unitários

Este projeto é uma calculadora desenvolvida em JavaScript com foco em testes unitários, validações e tratamento de erros.

## Funcionalidades

A aplicação realiza as seguintes operações:

- Soma
- Subtração
- Multiplicação
- Divisão
- Potência
- Raiz quadrada

## Tratativas

Foram implementadas tratativas com try-catch para a correta resposta

## Validações

Foram implementadas validações para garantir o correto funcionamento do sistema:

- Verificação de campos obrigatórios
- Validação de entrada numérica
- Tratamento de divisão por zero
- Tratamento de raiz de número negativo
- Uso de erros com throw

## Testes

Os testes foram realizados com Jest e cobrem:

- Casos de sucesso
- Casos de erro com throw
- Validação de entradas inválidas
- Casos limite

## Estrutura do projeto

/services  
  exercicios.js  

/controllers  
  controller.js  

/tests  
  service.test.js  

## Tecnologias

- JavaScript
- Node.js
- Jest
- Express

## Objetivo

O objetivo do projeto é praticar lógica de programação, testes unitários e tratamento de erros em uma API simples.
